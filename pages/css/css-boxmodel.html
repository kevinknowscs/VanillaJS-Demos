CSS Box Model

Introduction

The CSS box model consists of margins, borders, padding, and content. The top, left, bottom, and right values can be set independently, or they can be set with a single setting using a shortcut specifying one or more values.

When shortcuts are used, the top value is specified first, and then proceeds clockwise around the box; i.e., top, then right, then bottom, then left. See below for more details.

Box Model Properties

CSS Property Description
Margins Margins are used to push the box border away from the surrounding content
margin‑top: <value>; The top margin
margin‑left: <value>; The left margin
margin‑bottom: <value>; The bottom margin
margin‑right: <value>; The right margin
Margin Shortcuts Specify all margins in a single setting
margin: 20px, 15px, 10px 5px; Top is 20px; Right is 15px; Bottom is 10px; Left is 5px
margin: 20px, 15px, 10px; Top is 20px; Right is 15px; Bottom is 10px; 15px. Since the left margin value was omitted, the value of the right margin is used for both left and right.
margin: 20px, 15px; Top is 20px; Right is 15px; Bottom is 20px; Left is 15px. Since only two values were specified, the first value is used for the top and bottom, and the second value is used for the left and right.
margin: 20px; All four margins are set to 20px. Since only one value is specified, the value is used for all four settings.
Padding Padding pushes the content away from the box border. The padding settings are analagous to the margin settings.
padding‑top: <value>; The top padding
padding‑left: <value>; The left padding
padding‑bottom: <value>; The bottom padding
margin‑right: <value>; The right padding
Padding Shortcuts Specify all paddings in a single setting
padding: 20px, 15px, 10px 5px; Same as margins. See above.
padding: 20px, 15px, 10px; Same as margins. See above.
padding: 20px, 15px; Same as margins. See above.
padding: 20px; Same as margins. See above.
